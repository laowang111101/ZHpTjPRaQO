# 基于Hadoop的租房数据分析系统 python565

## 项目概述

本项目是一个基于Hadoop框架的租房数据分析系统，运用大数据技术对租房市场信息进行高效存储和计算，旨在为用户提供全面、准确的数据分析服务。

## 技术栈

- **后端：** Python 3.7 + Flask + MySQL 5.7
- **前端：** Vue
- **数据采集：** 爬虫技术
- **大数据处理：** Hadoop

## 功能模块

### 系统角色

- 管理员

### 主要功能

- 系统首页：展示系统概览和导航。
- 个人中心：用户个人信息管理。
- 用户管理：对系统用户进行管理操作。
- 房屋信息管理：对房屋租赁信息进行管理。
- 租房数据管理：分析处理租房数据。
- 系统管理：负责系统整体配置和维护。

## 系统架构

- **Hadoop框架：** 利用HDFS进行分布式存储，采用MapReduce进行分布式计算。

## 运行环境

- Python 3.7及以上版本
- Flask框架
- MySQL 5.7数据库
- Vue前端框架
- Hadoop环境

## 部署步骤

1. 安装Python 3.7及Flask
2. 配置MySQL 5.7数据库
3. 前端Vue框架搭建
4. Hadoop环境部署
5. 爬虫程序部署
6. 系统后端及前端服务启动
